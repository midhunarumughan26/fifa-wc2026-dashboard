 FIFA World Cup 2026 — Player Performance Dashboard
 
 Overview
End-to-end analytics pipeline built on 54,600 player-match 
records from FIFA World Cup 2026.

 Tools
- MySQL — Star schema database (3 tables)
- Python (pandas, SQLAlchemy) — ETL pipeline
- Power BI — Interactive dashboard with DAX measures

 Schema
- dim_players — 1,248 unique players
- dim_matches — 1,050 unique matches
- fact_performance — 54,600 records, 75 attributes

 Dashboard Highlights
- 6 KPI cards with accent colours
- Donut, Area, Pie, Scatter, Stacked bar, Grouped bar charts
- 3 interactive slicers (Team, Position, Stage)
- Single-page 1920×1080 dark theme layout

 Key Insights
- Qatar led all teams in xG overperformance
- Midfielders cover the most distance per match
- Overall pass completion: 80.67% across all teams
- Shot conversion rate: 12.39%
